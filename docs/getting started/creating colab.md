# Creating a Google Colab file

## Installation
To start with we need to to add Google Colab to your Drive<br/><br/>

Click on New
![Google Colab installation](../public/colabinstall/new.png)

Then click on more > connect more apps
![Google Colab installation](../public/colabinstall/connect.png)

In the search bar type 'Colab' or 'Colaboratory'
![Google Colab installation](../public/colabinstall/search.png)

Finally, click on the install button
![Google Colab installation](../public/colabinstall/install.png)

You should now see 'Google Colaboratory' as an option under new - Success!
![Google Colab installation](../public/colabinstall/success.png)


## Making the file
Click on 'Google Colaboratory' under more
![Google Colab installation](../public/colabinstall/success.png)


You should see a similar screen to the one below!
![Google Colab installation](../public/colabinstall/loaded.png)

Now head over to [Battleships](../battleship.md) to get started!
